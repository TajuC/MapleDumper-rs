# MapleDumper sample patterns
#
# Line forms (all accepted):
#   Name = AA BB ?? CC
#   Name: 0xAA ?? CC
#   Name AA ?? CC
#
# Wildcards: ? or ??     Comments: text after ; or #
# Architecture sections: #32BIT / #64BIT select which block to load.
# Category sections:     [name] groups the following symbols in offsets.h.
#
# Name suffixes control resolution:
#   *_PTR  -> resolve a RIP-relative load or rel32 jmp/call target (emits the target RVA)
#   *_CALL -> treat the match as a call and resolve the (nested) call target
#   *_OFF  -> extract a struct member displacement (emits the raw offset, not an RVA)
#   (none) -> emit the match address itself as an RVA
